# Divider

**File:** `src/components/Divider.tsx`
**Dependencies:** design tokens

## What It Is

Horizontal line separator for list items and content sections.

## Props

| Prop | Type | Default | Notes |
|------|------|---------|-------|
| inset | number | 0 | Left margin (for list item dividers that don't span full width) |

## Accessibility

- `accessible={false}` — purely decorative

## Stories

1. Full-width divider
2. Inset divider (simulating list row separator)

## Used On

Restaurant View item list, profile rating history, item detail rating cards, settings rows
